Scoring Engine Frontend

Created a custom scoring webpage for a cyber security red vs. blue team competition, built in React.js

Project Background

Cyber Defense Techniques Course @ RIT – Fall 2025

The WarGames competition is a red versus blue cybersecurity competition themed around the popular movie WarGames. The blue team was split into three competing groups: Alpha, Bravo, and Charlie. Each team would gain points for each service they kept up.

I was responsible for designing and creating a front-end web application that displayed team scores. I chose React.js and Vite as the development environment, as they would allow me to gain more experience with JavaScript while providing useful libraries for CSS through Vite.

View on GitHub

Issues & Conflict

One issue I encountered was task prioritization. Before finishing the barebones functionality of the site, I experimented with different ideas to make the page unique. I was able to script an animation that zoomed in and out of locations on the world map, but it did not make it to the final version as I had not left myself enough time to finish it. Ultimately, I learned to focus on the minimum viable product before adding feature enhancements.
